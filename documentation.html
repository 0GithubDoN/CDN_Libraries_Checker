Getting Started

Welcome to the CDN Library Manager! This tool helps you validate, manage, and update your CDN dependencies with ease. Here's a quick overview of what you can do:

Validate URLs

Check accessibility, CORS support, and content types

Update Libraries

Find and update to the latest versions

Process Files

Scan and update CDNs in your files

Customize

Configure validation options and preferences

URL Validator

1

Navigate to URL Validator

Click the "URL Validator" tab in the main navigation

2

Enter Your URLs

Paste your CDN URLs in any format:

  • Full HTML script tags
  • Direct URLs
  • Multiple URLs (one per line)
3

Validate

Click "Validate URLs" to check:

Accessibility
CORS Support
Content Type
SSL Certificate

File Processor

1

Upload Files

Drag and drop your files or click to select. Supported formats:

.html .js .php .jsx .vue
2

Process Files

Click "Process & Update Files" to:

  • Extract CDN URLs automatically
  • Check for available updates
  • Download updated versions
